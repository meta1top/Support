# Support Documentation Index

**Type:** Monorepo with 6 parts

## Project Overview

Support 是一个基于 NestJS、Next.js 和 React 构建的企业级全栈开发 Monorepo 项目，提供完整的后端服务、前端应用和 UI 组件库。项目采用 Monorepo 架构，使用 pnpm workspace 管理多个应用程序和共享库。

## Project Structure

This project consists of 6 parts:

### Server-Demo (server-demo)

- **Type:** Backend (NestJS)
- **Location:** `apps/server-demo/`
- **Tech Stack:** NestJS 11 + TypeORM + MySQL + Redis + Nacos
- **Entry Point:** `apps/server-demo/src/main.ts`

### Web-Design (web-design)

- **Type:** Web (Next.js)
- **Location:** `apps/web-design/`
- **Tech Stack:** Next.js 16 + React 19 + Tailwind CSS
- **Entry Point:** `apps/web-design/src/app/layout.tsx`

### Web-Editor (web-editor)

- **Type:** Web (Next.js)
- **Location:** `apps/web-editor/`
- **Tech Stack:** Next.js 16 + React 19 + Tiptap
- **Entry Point:** `apps/web-editor/src/app/layout.tsx`

### Libs (libs)

- **Type:** Library (NestJS Modules)
- **Location:** `libs/`
- **Tech Stack:** NestJS 11 + TypeScript
- **Entry Point:** Various module entry files

### Design (design)

- **Type:** Library (React Components)
- **Location:** `packages/design/`
- **Tech Stack:** React 19 + Radix UI + Tailwind CSS
- **Entry Point:** `packages/design/src/index.ts`

### Editor (editor)

- **Type:** Library (React Components)
- **Location:** `packages/editor/`
- **Tech Stack:** React 19 + Tiptap
- **Entry Point:** `packages/editor/src/index.ts`

## Cross-Part Integration

- **后端 ↔ 前端**: 通过 REST API 通信（HTTP/HTTPS，JSON 格式，JWT Token 认证）
- **前端 ↔ 组件库**: 通过 npm workspace 包导入使用
- **后端 ↔ 库**: 通过 NestJS 模块系统依赖注入
- **共享资源**: 类型定义（`libs/types/`）和国际化文件（`locales/`）前后端共享

## Getting Started

### Prerequisites

- Node.js >= 18
- pnpm >= 8
- Redis >= 6.0（用于缓存和会话）
- MySQL >= 8.0（可选，用于数据库）
- Nacos >= 2.0（可选，用于配置管理）

### Setup

```bash
# 克隆仓库
git clone <repository-url>
cd support

# 安装依赖
pnpm install
```

### Run Locally

```bash
# 启动后端服务（端口 3100）
pnpm run dev:server

# 启动设计系统展示（端口 5700）
pnpm run dev:web-design

# 启动编辑器展示（端口 5701）
pnpm run dev:web-editor
```

### Run Tests

```bash
# 运行单元测试
pnpm run test

# 监听模式运行测试
pnpm run test:watch

# 生成覆盖率报告
pnpm run test:cov
```
